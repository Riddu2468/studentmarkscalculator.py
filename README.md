# Student Marks Calculator

A simple Python program that calculates the total marks and percentage of a student.

## Features
- Takes marks as input
- Calculates total marks
- Calculates percentage
- Displays results clearly

## Concepts Used
- Variables
- User Input
- Arithmetic Operations
- Print Statements

## How to Run

1. Download the project
2. Open in Thonny or VS Code
3. Run the Python file

## Sample Output

Enter English marks: 80
Enter Hindi marks: 75
Enter Maths marks: 90

Total Marks: 245
Percentage: 81.67%
